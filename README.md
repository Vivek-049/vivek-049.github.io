# Portfolio Website

A modern, responsive portfolio website showcasing my projects, skills, and experience in robotics and software development.

## Features

- **Responsive Design** - Works perfectly on desktop, tablet, and mobile
- **Dark Theme** - Modern dark UI with smooth animations
- **GitHub Integration** - Automatically fetches and displays your latest repositories
- **Smooth Animations** - Engaging scroll animations and transitions
- **Fast & Lightweight** - Pure HTML, CSS, and JavaScript (no frameworks)

## Sections

- **Home** - Hero section with introduction
- **About** - Background and expertise
- **Projects** - Automatically populated from GitHub repos
- **Skills** - Technical skills and technologies
- **Contact** - Multiple ways to get in touch

## Deployment

This site is designed to be hosted on GitHub Pages.

### Setup Instructions

1. Create a new repository named `vivek-049.github.io`
2. Push this code to the repository
3. Enable GitHub Pages in repository settings
4. Your site will be live at `https://vivek-049.github.io`

## Customization

To customize the portfolio:

1. **Update Personal Info** - Edit `index.html` to change name, bio, and contact details
2. **Modify Skills** - Update the skills section in `index.html`
3. **Change Colors** - Edit CSS variables in `styles.css` (`:root` section)
4. **Add Projects** - Projects are automatically fetched from your GitHub profile

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- GitHub API
- Font Awesome Icons
